Exercícios Docker - PB Compass UOL - ABR 2025

Este projeto contém a solução do exercício 02, com foco em práticas de containers usando Docker.

O conteúdo desta página está hospedado em um container NGINX, a partir de um volume que monta a pasta html do projeto.

A página está acessível sempre que o container estiver em execução.
